## v0.18.0 - 2026-02-06
### Added
* New shimdatabase tag (TAG_OS_CODENAME)
* Optional support for generating a marker file
* Support for getting product info for a system. Ex: Dell device
* Support for generating a report summary after each collection
* Support for illumos OS
* Support for remote hosted toml collections
### Changed
* Output structure for Windows BITs artifact
* Updated Schedule Task output format
### Fixed
* Issue where artemis sometimes would fail to parse Windows 11 24H2 ESE files
* Improved BITs parsing support
* Bug where artemis would include empty timestamps in rawfilelisting timeline when $INDX recovery was enabled
* Major bug where the rawfilelisting did not display accurate FILENAME timestamps
* Better handling for parsing eventlogs with missing provider dlls
* Bug where artemis would not correctly timeline loginitems in app bundles
### ArtemisApi
* Support for extracting RDP events from Windows EventLogs
* Support for parsing Program Compatability Assist (PCA) text files
* Support for parsing Firefox form history
* Additional tests, code cleanup, and improved documentation
* Support for extracting Defender quarantine events from EventLogs
